# MPRA_Analyzer
This is a toolkit for MPRA data analysis: Data quality control, Activity, Different elements activity, Motif analysis, etc.

### Package/tools dependency

R >=3.6.1

DESeq2 1.34.0

dplyr 1.1.2 

SummarizedExperiment 1.24.0

ggplot2 3.4.2

jmuOutlier 2.2

## Installation

Download the code and source it!
